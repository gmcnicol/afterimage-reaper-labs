# AGENTS

Mission: prove or kill future Afterimage ideas as cheaply as possible.

This repository is a laboratory.

Do not optimise for architecture, productisation, UI polish, packaging, or distribution.

Optimise for learning.

Every experiment must:

- State a hypothesis.
- Produce evidence.
- End with a decision.

Preferred technologies:

- Lua
- JSFX
- Reaper automation
- Reaper video processor
- MIDI
- OSC
- FFmpeg

Avoid introducing external frameworks unless required.

The goal is not to build Afterimage inside Reaper.

The goal is to determine which ideas deserve to graduate into Afterimage core.
