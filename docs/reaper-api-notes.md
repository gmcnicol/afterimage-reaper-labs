# Reaper API Notes

These notes capture the minimum ReaScript calls used by the LAB-001 importer.

Official reference: https://www.reaper.fm/sdk/reascript/reascripthelp.html

## Track setup

- `reaper.CountTracks(0)` counts project tracks.
- `reaper.GetTrack(0, index)` fetches a track by zero-based index.
- `reaper.InsertTrackAtIndex(index, true)` inserts a track.
- `reaper.DeleteTrack(track)` deletes a previously owned lab track during rebuild.
- `reaper.GetSetMediaTrackInfo_String(track, "P_NAME", name, true)` sets a track name.
- `reaper.GetSetMediaTrackInfo_String(track, "P_EXT:lab001-owner", value, true)` marks LAB-001-owned tracks.
- `reaper.SetOnlyTrackSelected(track)` makes one track the current selected track.

## Media placement

- `reaper.SetEditCurPos(time, false, false)` moves the edit cursor.
- `reaper.InsertMedia(path, 0)` inserts media on the current track.
- `reaper.CountTrackMediaItems(track)` and `reaper.GetTrackMediaItem(track, index)` inspect track items.
- `reaper.SetMediaItemInfo_Value(item, "D_POSITION", seconds)` sets item position.
- `reaper.SetMediaItemInfo_Value(item, "D_LENGTH", seconds)` sets item length.
- `reaper.GetActiveTake(item)` fetches the active take.
- `reaper.GetSetMediaItemTakeInfo_String(take, "P_NAME", name, true)` sets take name.

## Project grid and tempo

- `reaper.GetSetProjectGrid(0, false, 0, 0, 0)` reads the current project grid division.
- `reaper.TimeMap2_GetDividedBpmAtTime(0, 0)` reads the project tempo used for grid-cut suggestions.
- `reaper.Master_GetTempo()` is a fallback tempo source.

## Markers, regions, and project ownership

- `reaper.AddProjectMarker2(0, true, start, finish, name, -1, 0)` adds inspectable scene regions.
- `reaper.AddProjectMarker2(0, false, position, 0, name, -1, 0)` adds inspectable grid-cut markers.
- `reaper.CountProjectMarkers(0)` and `reaper.EnumProjectMarkers3(0, index)` enumerate prior generated regions/markers.
- `reaper.DeleteProjectMarkerByIndex(0, index)` removes prior generated regions/markers during rebuild.
- `reaper.SetProjExtState(0, section, key, value)` records LAB-001 project ownership metadata.
- `reaper.GetProjExtState(0, section, key)` checks whether prior LAB-001 markers were generated by this importer.

## First-pass constraints

The importer creates a source-video substrate with optional supporting media and inspectable scene/grid-cut regions. It does not yet build routing, video processors, automation lanes, semantic scene understanding, time-stretching, or Afterimage archive compatibility beyond the minimal lab contract.
