# Roadmap Alignment

Afterimage Reaper Labs exists to reduce uncertainty before ideas graduate into Afterimage core.

## Current lab order

1. Import a small Afterimage archive into a Reaper timeline.
2. Write generated values to Reaper automation envelopes.
3. Use those envelopes to drive video parameters.
4. Render evidence and decide whether the behavior graduates.

## Graduation standard

An idea deserves Afterimage attention only when the lab has produced:

- A clear hypothesis.
- A runnable prototype.
- Inspectable evidence.
- A decision.

## Non-goals

Do not turn this repo into a standalone product, plugin suite, or polished authoring app. Keep the work biased toward learning.
